Home Page Info


This is the information page of the 'Home Page', the start of the application.

We call this the 'Setup Phase'.

  • This is the page where we enter all the basic details.
  • The user enters the name of the file where the real-time data recorded from the camera should be stored.
  • The user then enters the name of all the tools with the corresponding ArUco marker IDs that he wishes to use during the DIY process.
  • The user can add or delete as many tools as he wishes, using the button and the button respectively.

Example run:

For this example run, we name the file ’Example run’ and add the tools:

  • Hammer with ArUco marker ID 1
  • Cutter with ID 0
  • Saw with ID 2
  • Measuring Tape with ID 3

We then click on the ‘To File Creation’ button to proceed to the next phase, the ‘File Creation Phase’.
