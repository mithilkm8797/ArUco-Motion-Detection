Visualization Info


This is the information page of the 'Visualization Page'.

This is also the 'Visualization Phase'.

  • In this phase, we visualize the time series data and plot the tool usage graphs for the entire timeline of the DIY process. We also display the tool usage order along with the duration statistics.

    The three different graphs are:
    1. The combined tool usage plot of all the tools
    2. The separated tool usage plots of each tool.
    3. The KDE plot of all the tools - A kernel density estimate (KDE) plot is a method for visualizing the distribution of observations in a dataset, analogous to a histogram. KDE represents the data using a continuous probability density curve in one or more dimensions.


    In the following images below we can see the tool usage statistics followed by the three graphs one after the other in order.

    Tool usage Statistics: Shows a table summarizing the order in which the tools were used, the start and end usage time of each tool, and the duration of usage in seconds.



    Combined Plot: A plot where the tool usage visualization of all the tools are combined in one single graph. 0.0 on the Y-axis indicates the tool being at rest, while 1.0 indicates the tool being used. 0.5 does not mean anything.



    Separated plot: A plot where the tool usage visualization of each tool is plotted separately and stacked on top on each other. 0.0 on the Y-axis indicates the tool being at rest, while 1.0 indicates the tool being used. 0.5 does not mean anything.



    KDE Plot: This plot shows the density of the tool usage of each tool. The peak indicates that the corresponding tool was used for the maximum time during the entire timeline of the DIY process.
