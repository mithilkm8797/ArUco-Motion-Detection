File Creation Info


This is the information page of the 'File Creation Page'.

There are two phases in this page.
The 'File Creation Phase' and the 'Preparation Phase'.

File Creation Phase:
We deal with the file creation aspects in this phase.

  • If the file does not exist, a new file will be created (see image below).
    The user can click on the 'Next >' button to proceed to the 'Preparation Phase'.


  • If the file already exists, the user is given an option to either append the data that will be generated in the current run to the existing file by clicking the ‘Append > ’ button or delete the existing file and start with a new file by clicking the ‘Delete > ’ button (see image below). Clicking either of the buttons takes the user to the next phase ('Preparation Phase').

Preparation Phase:
We deal with the tool confirmation and preparation (tagging the tools with ArUco markers) aspects in this phase.

  • From the 'Setup Phase' we have:

    • Hammer with ArUco marker ID 1
    • Cutter with ID 0
    • Saw with ID 2
    • Measuring Tape with ID 3

    The user must confirm the tool list from the table displayed (see image below). If the tool list does not match, the user has the option to go back to the 'Setup Phase' to correct it.


  • The link in the red, can be used to generate and print ArUco markers for free.
    Clicking on it should redirect the user to https://chev.me/arucogen/.
    The user must enter the dictionary type, marker ID, and it's desired dimensions, for it to be generated.
