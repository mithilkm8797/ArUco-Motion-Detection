ArUco Motion Detection

Detection Info


This is the information page of the 'Detection Page'.

This is also the 'Detection Phase'.

NOTE:

For the cases where no data was added (no detections were made), please go to the console of your IDE (the application you used to run the application) and proceed with the further instructions from there.

  • The application starts off by informing the user about the minimum requirement/constraint of the prototype, about the markers attached to the tool being visible to the camera at all times when the tool is at rest, throughout the DIY process (see image below).

    The user must click on the 'Understood' button to proceed.


  • The user can then click on the ‘Start Video Stream’ button to initiate the capturing process/start the video stream for detection (see image below).


  • A pop-up informing the user necessitating the clicking of the 'x' icon of the video stream window to end the capturing process appears. The popup also informs that the user now has to move to the tool usage area to start using the tools (see image below).

    The user must click on the 'Understood' button to proceed. Note that the window can also be closed by right-clicking on the detection window and choosing to close it.


  • The detection/capturing window shows the tools being detected (see image below).


  • The console (of the IDE which is being used to run the web application) shows the detection information such as the detected tool, it's ID, and it's X and Y co-ordinates (see image below):


  • The user must then close the capturing window by clicking on the 'X' button. This ends the detection process.

    The user must click on the 'To Visualizations > ' button to proceed to the 'Visualization Phase'.
